**Test Project**

---

### Installation

To install the dependencies, run the following command:

```bash
npm install
```

### Linter

To lint the project files, use the following command:

```bash
npm run lint
```

### Tests

To run the tests, execute the following command:

```bash
npm test
```

### Run Project

To start the project, run the following command:

```bash
npm start
```
